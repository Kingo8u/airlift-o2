# AirLift O₂ — Website

Landing page for AirLift O₂, a ceiling-mounted track system that keeps medical oxygen tubing off the floor.

## Stack
- Static HTML/CSS/JS (single page)
- Hosted on Vercel

## Local Preview
Open `public/index.html` in a browser.

## Deploy
Connected to Vercel — push to `main` to deploy automatically.
